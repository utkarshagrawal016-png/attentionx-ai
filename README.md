# AttentionX AI - Automated Content Repurposing Engine

## Problem

Long videos contain valuable insights but are not easily consumable.

## Solution

AttentionX AI converts long videos into short, engaging clips automatically.

## Features

* Upload video
* Detect key moments (AI-based / simulated)
* Generate short clips
* Simple UI using Streamlit

## Tech Stack

* Python
* Streamlit
* MoviePy
